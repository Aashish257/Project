# Accio Projects Portfolio

This repository contains a collection of web applications built as part of the Accio development program. Each folder represents an independent project focusing on different frontend technologies, APIs, layouts, and responsive designs.

---

## 📂 Completed Projects Directory

### 1. Restaurant Website
* **Description**: A premium, responsive landing page for a restaurant website. Includes an interactive menu display, customer testimonials, and contact forms.
* **Tech Stack**: HTML5, Vanilla CSS3 (Flexbox/Grid), JavaScript.
* **🔗 Live Link**: [Open Restaurant Website](https://fantastic-sunburst-e83b19.netlify.app/)

### 2. Todo List
* **Description**: A task manager application allowing users to add, complete, and filter tasks. Uses browser local storage to persist data.
* **Tech Stack**: HTML5, CSS3, JavaScript (DOM Manipulation, LocalStorage).
* **🔗 Live Link**: [Open Todo List](https://tangerine-ganache-009dde.netlify.app/)

### 3. Multi-API Data Display
* **Description**: An interactive landing page that queries data from multiple public endpoints and structures the responses in stylized grid grids.
* **Tech Stack**: HTML5, CSS3, JavaScript (Fetch, Promise).
* **🔗 Live Link**: [Open Multi-API Data Display](https://dancing-tarsier-4f3f64.netlify.app/)

### 4. Retrieve Timezone Information
* **Description**: Translates coordinate points into comprehensive timezone details. Uses browser Geolocation coordinates and the Geoapify API.
* **Tech Stack**: HTML5 Geolocation API, Geoapify Reverse Geocoding API, Vanilla JS.
* **🔗 Live Link**: [Open Timezone Information]https://golden-khapse-b23770.netlify.app/)

### 5. Retrieve and Display User Information Using IP Address
* **Description**: A location dashboard that resolves client public IP addresses, displays location coordinates, embeds a Google map, ticks a timezone-localized clock, and filters local postal code branch locations.
* **Tech Stack**: HTML5, Vanilla CSS3, JavaScript, `ipinfo.io` Geolocation API, Google Maps Embed, Post Office API.
* **🔗 Live Link**: [Open IP Geolocation Dashboard](https://superb-cobbler-8aa463.netlify.app/)

### 6. Crypto Market Data
* **Description**: A crypto tracking dashboard that pulls coin statistics, market capitalization, prices, and volumes.
* **Tech Stack**: HTML5, CSS3, JavaScript, Crypto API integration.
* **🔗 Live Link**: [Open Crypto Market Dashboard](https://dulcet-meringue-b96959.netlify.app/)

## ⚙️ How to Deploy & View Projects
1. Push this root repository to your GitHub.
2. In your GitHub repository settings, navigate to **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch** and choose `main` (or `master`) as your source, then click **Save**.
4. After a minute, replace `your-github-username` and `your-repository-name` in the links above with your actual details to access your live portfolio!
